NppTags
=======

NppTags is a [Universal CTags](https://ctags.io) plug-in for [Notepad++](https://notepad-plus-plus.org/).

The documentation of this plug-in can be found at [Read the Docs](http://npptags.readthedocs.org/). This includes:

* Instructions [how to install](http://npptags.readthedocs.org/en/latest/installation.html)
* Instructions [how to use](http://npptags.readthedocs.org/en/latest/how-to-use.html)
* Instructions [how to build](http://npptags.readthedocs.org/en/latest/compile.html)
* A complete [version history](http://npptags.readthedocs.org/en/latest/history.html)
